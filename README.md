# Jacques McDermid Heyns – Virtual Portfolio Website

## Overview

This repository contains my **Virtual Resume Portfolio Website**, developed as part of the **CMPG 323 Portfolio of Evidence Assignment**.

The purpose of this project is to present my curriculum vitae in the form of a professional, responsive website hosted using **GitHub Pages**. The website showcases my professional profile, qualifications, experience, skills, and portfolio in an accessible and user-friendly format.

---

## Website Features

The portfolio website includes the following sections:

- Home
- About Me
- Resume
- Projects
- Skills
- Contact Information

The website has been designed with an emphasis on:

- Responsive web design
- Clear navigation
- Professional presentation
- Accessibility
- Clean user interface
- Mobile compatibility
